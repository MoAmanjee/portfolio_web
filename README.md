# Portfolio Website Example

A beautiful, professional portfolio website example built for Webs4U to showcase their work.

## Features

- Modern, clean design
- Fully responsive (mobile, tablet, desktop)
- Smooth animations and transitions
- Contact form
- Easy to customize

## How to Use

1. **Customize the Content:**
   - Edit `index.html` to change text, names, and information
   - Update colors in `styles.css` if needed
   - Add your own images or replace gradient backgrounds

2. **Deploy:**
   - Upload to any web hosting service
   - Or use GitHub Pages, Netlify, or Vercel for free hosting

3. **Link from Webs4U:**
   - Once deployed, add the URL to your Webs4U portfolio section
   - Update the portfolio items in your main website to link to this example

## Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
}
```

### Add Real Images
Replace the gradient backgrounds in portfolio items with actual images:
```html
<div class="portfolio-image" style="background-image: url('your-image.jpg');">
```

### Update Contact Info
Edit the contact section in `index.html` with real contact details.

## Deployment Options

### GitHub Pages
1. Create a new repository
2. Upload these files
3. Enable GitHub Pages in repository settings
4. Your site will be live at `username.github.io/repository-name`

### Netlify
1. Go to netlify.com
2. Drag and drop this folder
3. Your site is live instantly!

### Vercel
1. Go to vercel.com
2. Import from GitHub or upload folder
3. Deploy with one click
